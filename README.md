# 🌐 Personal Portfolio Website

A responsive personal portfolio website built using **HTML, CSS, and JavaScript** to showcase my skills, education, and projects with a clean and modern user interface.

---

## 🚀 Features

* Responsive design (Mobile + Desktop)
* Clean and modern UI
* Interactive navigation bar
* Hero section with call-to-action
* Skills section
* Projects showcase
* Contact form with basic JavaScript functionality
* Smooth hover effects and transitions

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

## 📂 Project Structure

```
portfolio/
│── index.html   # Main file (HTML + CSS + JS in one file)
```

---

## ▶️ How to Run

1. Download or clone the repository
2. Open `portfolio.html` in any browser
3. Done ✅
